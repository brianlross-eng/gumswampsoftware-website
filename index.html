About

Gum Swamp Software (GSS) is an independent software company based in Beaufort, South Carolina, building open-source tools and applications. Named after the blackwater cypress swamps of the Carolina coast.

Lazarus

A free PyPI-compatible proxy that automatically fixes Python packages broken by Python 3.14 incompatibilities. Fetches packages from PyPI, applies fixes, and serves them transparently.

500k+ Packages processed
88% Success rate
22 Auto-fix types

Quick Start

pip install --index-url https://lazaruspy.org/simple/ your-package

Or configure globally in pip.conf:

[global]
index-url = https://lazaruspy.org/simple/
extra-index-url = https://pypi.org/simple/

About

Lazarus

A PyPI-compatible proxy that automatically fixes Python packages broken by Python 3.14 incompatibilities. Fetches packages from PyPI, applies fixes, and serves them transparently—no configuration needed.

500k+ Packages processed
88% Success rate
22 Auto-fix types

Quick Start

Replace your PyPI index with Lazarus:

pip install --index-url https://lazaruspy.org/simple/ your-package

Or configure globally in pip.conf:

[global]
index-url = https://lazaruspy.org/simple/
extra-index-url = https://pypi.org/simple/

Llamaste

A bootable Linux OS where a large language model runs as PID 1. Open-source, Apache 2.0 licensed. Run LLMs on bare metal with zero configuration.

Llamaste brings the power of modern LLMs to any x86-64 machine. Boot from USB, start generating, and forget about complex setup procedures. Perfect for development, testing, or running your own AI assistant.

Features

  • Pre-configured LLM runtime (PID 1)
  • Simple web interface for interaction
  • Model swapping without reinstallation
  • Lightweight, runs on minimal hardware
